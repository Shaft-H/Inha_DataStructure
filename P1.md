## (a) Sorting Algorithm with Time Complexity $O(n \log n)$

**Thought Process:**  
We are given that the running time is  
$ T(n)= c\, n \log n, $
and that $T(N)$ is known for some $N$. In particular, we have  
$$ c = \frac{T(N)}{N\log N}. $$  
Thus, for any input size $n$,  
$$ T(n)= T(N)\,\frac{n \log n}{N\log N}. $$

**Answer:**  
The general formula is  
$$ T(n)= T(N)\,\frac{n\log n}{N\log N}. $$  
For example, if $T(1000)=1$ millisecond, then to estimate $T(1\,000\,000)$, note that:  
- $n/N=1\,000\,000/1000 = 1000$, and  
- $\frac{\log(1\,000\,000)}{\log(1000)}=\frac{6\log(10)}{3\log(10)}=2$.  
Thus,  
$$ T(1\,000\,000)=1\,\text{ms} \times 1000 \times 2 = 2000\,\text{ms} = 2\,\text{seconds}. $$

## (b) Quadratic Algorithm (Assuming $T(n)= c\,n^2$)

**Thought Process:**  
Even though the statement reads “quadratic algorithm,” the formula is given as $T(n)= c\, n^2$ (note: the exclamation mark in the statement appears to be punctuation). We are told that for $N=100$ data items, $T(100)=1$ millisecond. So,  
$$ c = \frac{T(100)}{100^2} = \frac{1\,\text{ms}}{10000} = 0.0001\,\text{ms}. $$  
Then, for $n=5000$,  
$$ T(5000)= 0.0001\,\text{ms} \times (5000)^2 = 0.0001 \times 25 \times 10^6\,\text{ms} = 2500\,\text{ms} = 2.5\,\text{seconds}. $$

**Answer:**  
The time required to process 5000 items is approximately **2.5 seconds**.

## (c) Algorithm with Time Complexity $T(n)= c\,f(n)$

We are given that processing 1000 data items takes 10 seconds. We need to find the time for 100,000 items in two cases.

### Case 1: $f(n)= n$

**Thought Process:**  
For $n=1000$,  
$$ c = \frac{10}{1000} = 0.01\,\text{seconds per item}. $$  
For $n=100\,000$,  
$$ T(100\,000)= 0.01 \times 100\,000 = 1000\,\text{seconds}. $$

**Answer (i):**  
$$ T(100\,000)= 1000\,\text{seconds} $$  
(which is approximately 16 minutes 40 seconds).

### Case 2: $f(n)= n^2$

**Thought Process:**  
For $n=1000$,  
$$ c = \frac{10}{1000^2} = \frac{10}{1\,000\,000} = 0.00001\,\text{seconds per } n^2. $$  
For $n=100\,000$,  
$$ T(100\,000)= 0.00001 \times (100\,000)^2 = 0.00001 \times 10^{10} = 100\,000\,\text{seconds}. $$

**Answer (ii):**  
$$ T(100\,000)= 100\,000\,\text{seconds} $$  
(which is approximately 27.8 hours).

## (d) Additional: Big-O Properties – True or False

We are given several statements about Big-O notation. Here are my evaluations:

1. **Rule of sums:**  
   **Statement:** $O(f + g) = O(f) + O(g)$  
   **Evaluation:** FALSE  
   **Correct Formula:** $O(f+g)=O(\max\{f,g\})$.  
   (Big-O notation does not support addition in a linear sense.)

2. **Rule of products:**  
   **Statement:** $O(f\cdot g) = O(f) \cdot O(g)$  
   **Evaluation:** TRUE  
   (In the sense that if $f(n)=O(h(n))$ and $g(n)=O(k(n))$, then $f(n)g(n)=O(h(n)k(n))$. However, one must be cautious with notation since Big-O is a set; the property is generally understood in this multiplicative manner.)

3. **Transitivity:**  
   **Statement:** If $g=O(f)$ and $h=O(f)$ then $g=O(h)$.  
   **Evaluation:** FALSE  
   **Correct Statement (Transitivity):** If $g=O(h)$ and $h=O(f)$, then $g=O(f)$.

4. **Expression Comparison 1:**  
   **Statement:** $5n+8n^2+100n^3 = O(n^4)$  
   **Evaluation:** TRUE  
   (Though $O(n^3)$ is a tighter bound, $n^4$ is an upper bound.)

5. **Expression Comparison 2:**  
   **Statement:** $5n+8n^2+100n^3 = O(n^2\log n)$  
   **Evaluation:** FALSE  
   **Correct Formula:** $5n+8n^2+100n^3 = O(n^3)$ because the cubic term dominates $n^2\log n$.

## (e) Prove that 
$$ T(n)= a_0 + a_1 n + a_2 n! + a_3 n^n $$
is $O(n^n)$.

**Thought Process and Proof Outline:**  
We must show that there exist constants $c$ and $n_0$ such that for all $n\ge n_0$,
$$ T(n)\le c\, n^n. $$
Observe that for all $n\ge 1$:  
- $a_0$ is constant, so $a_0\le a_0\, n^n$ (since $n^n\ge 1$).  
- $a_1 n\le a_1\, n^n$ because $n^n$ grows much faster than $n$.  
- $a_2 n! \le a_2 n^n$ since for $n\ge 1$, it is known that $n! \le n^n$.  
- $a_3 n^n$ is already in the form $a_3 n^n$.

Thus, for any $n\ge 1$,
$$ T(n)\le (a_0+a_1+a_2+a_3)n^n. $$
Let $c = a_0+a_1+a_2+a_3$ and $n_0=1$. Then the inequality holds for all $n\ge n_0$.

**Answer:**  
By the formal definition of Big-O, we have shown that  
$$ T(n)\le c\,n^n \quad \text{for all } n\ge 1, $$
which proves that  
$$ T(n)= O(n^n). $$
